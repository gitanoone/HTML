Script types

For this, scripts are written- programs that allow you to respond to user actions. Scripts come in two forms:

  • those that are running on the server, and the result of their execution comes to the browser to the user in a ready-made form. These are scripts written in PHP. CMS is written on it - content management systems.
  • those that run directly in the user's browser. These are scripts written in JavaScript. They are most often used to make the page more convenient and beautiful.
